# Changelog

All notable changes to TRTourPackage will be documented in this file.

## [1.0.0] - 2026-02-13

### Added
- Initial release of TRTourPackage platform
- Multi-language support for 12 languages:
  - English (en)
  - Türkçe (tr)
  - العربية (ar)
  - Deutsch (de)
  - Italiano (it)
  - Français (fr)
  - 中文 (zh)
  - 한국어 (ko)
  - 日本語 (ja)
  - Русский (ru)
  - Azərbaycan (az)
  - Español (es)
- Language selector component with localStorage persistence
- Complete translations for all user-facing pages:
  - Landing page
  - Request form
  - Traveler login/register
  - Traveler dashboard
  - Agency login
  - Agency dashboard
  - Offer creation
  - FAQ page
  - About page
  - Contact page
  - Verification page
  - Privacy Policy
  - Terms of Service
- Public tour request form
- Agency dashboard for viewing and managing requests
- Offer creation and submission system
- 24-hour offer window
- TÜRSAB verification system
- Admin panel for agency management
- Traveler authentication system
- Agency authentication system
- Direct contact system (WhatsApp integration)
- Rating and review system
- Responsive design with clean UI
- Cookie consent banner
- Database schema with Prisma ORM
- Demo seed data for testing

### Technical
- Next.js 14 with App Router
- TypeScript for type safety
- Prisma ORM with SQLite (dev) / PostgreSQL (prod)
- React Context API for language management
- Lucide React icons
- Custom design system with Manrope and Inter fonts
- API routes for all backend operations
- Server-side rendering for SEO

### Design
- Professional, trust-focused design
- Color scheme: #e53e3e (primary), #2d3748 (text), #718096 (gray)
- Typography: Manrope (headings), Inter (body)
- Container width: 1400px
- Fully responsive layout
- Accessibility-compliant components
